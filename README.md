# Kimia Mostadam – Portfolio

UX Researcher & Designer portfolio website, built for GitHub Pages.

## 🚀 How to Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `kimia-portfolio`)
2. Upload all files from this folder to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**

Your site will be live at: `https://YOUR-USERNAME.github.io/kimia-portfolio`
